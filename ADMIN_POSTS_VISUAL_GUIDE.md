## Grid Layout

Posts are displayed in a responsive grid:
- **Desktop**: 3-4 columns
- **Tablet**: 2 columns
- **Mobile**: 1 column

Each card has:
- Fixed width: minimum 350px
- Auto-fill to available space
- 20px gap between cards
- Consistent height with flex layout

## Color Scheme

### Primary Colors
- **Purple**: #667eea (buttons, links)
- **Green**: #27ae60 (success, active status)
- **Orange**: #e65100 (warnings, flagged)
- **Red**: #c62828 (removed, errors)
- **Blue**: #1976d2 (info, post types)

### Background Colors
- **White**: #ffffff (cards)
- **Light Gray**: #f5f5f5 (media previews)
- **Very Light Gray**: #f9f9f9 (poll backgrounds)

### Text Colors
- **Dark**: #333 (headings)
- **Medium**: #666 (body text)
- **Light**: #999 (metadata, dates)

## Responsive Behavior

### Desktop (1400px+)
- 3-4 post cards per row
- Full sidebar visible
- All features visible

### Tablet (768px - 1399px)
- 2 post cards per row
- Collapsible sidebar
- Compact view

### Mobile (< 768px)
- 1 post card per row
- Mobile menu
- Stacked layout
- Touch-friendly buttons

## Accessibility

- All buttons have hover states
- Color contrast meets WCAG AA standards
- Keyboard navigation supported
- Screen reader friendly labels
- Focus indicators visible

## Performance

- Lazy loading with pagination (50 posts per page)
- Images lazy loaded
- Smooth scrolling
- Optimized re-renders with React hooks
- Efficient database queries with indexes
